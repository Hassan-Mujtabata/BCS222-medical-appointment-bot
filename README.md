# 🏥 BCS222-medical-appointment-bot

> An AI-powered medical appointment booking assistant that fills and submits hospital forms on behalf of users — built for accessibility, targeting the elderly and those with physical impairments.

---

## 📌 Overview

This project was built for **BCS 222 – Programming Paradigms** at Canadian University Dubai. It combines a Streamlit patient registration form, Google Gemini AI browser automation, and Hylang (Lisp) eligibility logic to automate the process of booking a medical appointment at **Cleveland Clinic Abu Dhabi**.

The user simply fills in their details through the GUI. The app validates the input, saves it securely to a `.env` file, applies Lisp-based eligibility logic, and — if eligible — launches an AI agent that navigates the hospital website and submits the appointment form automatically.

## 🧠 Programming Paradigms Covered

| Paradigm | How It's Applied |
|---|---|
| **Imperative** | Python reads `.env` variables and issues step-by-step browser commands |
| **Object-Oriented** | `Agent` and `Browser` objects from `browser-use` encapsulate AI automation logic |
| **Functional** | Hylang (Lisp) pure functions handle eligibility decision-making in `logic.hy` |
| **Interactive** | Streamlit reacts to user input in real time with dynamic validation and feedback |

## 📦 Installation

**1. Clone the repository**
```bash
git clone https://github.com/Hassan-Mujtabaa/BCS222-medical-appointment-bot.git
cd BCS222-medical-appointment-bot
```

**2. Install dependencies**
```bash
pip install streamlit python-dotenv browser-use langchain-google-genai pydantic hy
```

**3. Set up your `.env.example`**

The `.env` file is auto-generated by the app when you submit the form. However, you must set your **Gemini API key** manually before running:

```bash
cp .env.example .env
```

Then add your key to `.env`:
```
GEMINI_API_KEY=your_google_gemini_api_key
```

> ⚠️ Never commit your `.env` file. It contains personal data and API keys. It is listed in `.gitignore`.

---

## 🚀 Running the App

```bash
streamlit run paradigms_project.py
```

The app will open at `http://localhost:8501`.

## 🖥️ How to Use

1. Select **Cleveland Clinic Abu Dhabi** from the sidebar
2. Fill in your patient details:
   - First & last name, date of birth, gender, email, phone
   - Emirates ID (optional)
   - First-time visit status
   - Medical specialty needed
   - Payment method (Insurance or Self-pay)
3. Click **Submit**
4. The app validates your input, saves it to `.env`, and runs the Lisp eligibility check
5. If eligible, the Gemini AI agent launches automatically, navigates the hospital website, and submits your appointment form

---

## 🔍 Eligibility Logic (Lisp)

The `logic.hy` file uses a pure Hylang function to determine whether the form should be submitted:

- User must be a **first-time visitor**
- Selected specialty must **not** be `"Other"`
- Payment method must be either `"Insurance"` or `"Self-pay"`

If any condition fails, the agent does not launch and the user is notified.

## 📋 Form Fields & Validation

| Field | Validation |
|---|---|
| First / Last Name | Required |
| Email | Must match valid email format |
| Phone | Must be `+971` followed by exactly 9 digits |
| Emirates ID | Must be exactly 15 digits (if provided) |
| Specialty | Required selection |
| Insurance Provider | Required if payment method is Insurance |
